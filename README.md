# kotlin-aoc-2025

Solutions for [Advent of Code 2025](https://adventofcode.com/2025) in Kotlin.

## About

This repository contains my solutions for the Advent of Code 2025 puzzles, implemented in Kotlin.

## Getting Started

### Prerequisites

- [Kotlin](https://kotlinlang.org/) (1.9+)
- [JDK](https://openjdk.org/) (17+)

### Running Solutions

Each day's solution can be run independently. Input files should be placed in the appropriate location for each day's puzzle.

## Project Structure

```
kotlin-aoc-2025/
├── src/           # Source files for each day
├── input/         # Puzzle input files
└── README.md
```
